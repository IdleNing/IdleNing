《指尖英雄》隐私政策

感谢您信任并选择我们的游戏《指尖英雄》。

我们高度重视您的个人信息与隐私保护，为保障您的合法权益，在您进入游戏前，请务必仔细阅读本隐私政策的全部条款，尤其注意以下重点内容：

权限申请仅为实现游戏核心功能：

  • 【存储权限】：用于保存游戏内截图 / 视频（支持社交分享）及本地存档数据（防止进度丢失），不用于存储无关文件；
  • 【位置权限】：仅当您使用 "附近玩家匹配"（若有此功能）或参与地域限定活动时需授权，您可拒绝，不影响其他功能使用。

实名认证为法定要求：

根据《关于进一步严格管理 切实防止未成年人沉迷网络游戏的通知》，您需提供真实身份信息完成实名认证，否则无法使用游戏服务；您的实名信息仅用于与公安权威平台比对，比对后不留存原始数据。

第三方 SDK 信息披露：

游戏接入第三方 SDK 以提供广告、统计及基础运行服务，各 SDK 收集的信息及隐私政策如下（请您重点关注）：

  • 穿山甲 SDK（广告服务）：获取 Android ID、OAID、IP 地址、MAC 地址、应用安装列表、传感器信息，隐私政策：https://www.pangle.cn/privacy
  • Gromore SDK（广告服务）：https://www.csjplatform.com/supportcenter/5879
  • 优量汇 SDK（广告服务）：获取 Android ID、IP 地址、应用安装列表、传感器信息，隐私政策：https://www.qq.com/privacy_sdk.htm
  • Tap SDK（账号登录 / 运营统计）：获取 Android ID，隐私政策：https://www.taptap.cn/privacy
  • Godot 引擎（游戏引擎组件）：获取 Android ID，无独立隐私政策（相关信息处理遵循本政策）

您勾选 "阅读并同意隐私政策"，即表示您已充分理解并同意本政策全部内容。

一、我们收集和使用的个人信息

我们仅收集为提供、优化游戏服务及保障账号安全所必需的信息，具体如下：

1.1 账号与身份标识信息：

当您登录游戏时，我们会收集您的Android ID、OAID（均为设备唯一标识），用于标记您的游戏账号、记录 CDK 兑换记录（防止重复使用），确保游戏正常运营；iOS 设备端将收集 IDFA（若您授权），用途同上。

1.2 实名认证信息：

根据法律法规要求，您需提供姓名、身份证号完成实名认证，信息通过 TDS 实名认证系统与公安权威数据平台比对，比对后立即删除原始信息，不存储。

1.3 设备与运营数据：

为优化游戏性能、保障账号安全及打击作弊，我们会收集您的设备名称、型号、操作系统版本、IP 地址、设备识别符（Android ID、OAID、IMEI，仅用于设备唯一性标识）、游戏崩溃记录、已安装应用列表（仅用于检测外挂）、游戏渠道来源、登录时间、游戏时长；

若您授权位置权限，仅收集粗略地理位置信息（如城市级别），用于地域活动推送（如 "同城赛事"），不获取精确 GPS 坐标。

1.4 权限申请说明：

  • 【存储权限】：实现截图保存、存档存储，拒绝后无法使用截图分享及本地存档功能，但可使用云端存档（若有）；
  • 【位置权限】：仅用于地域相关功能，拒绝后不影响游戏核心玩法；

您可通过设备【设置】-【应用管理】-《指尖英雄》调整权限授权状态。

二、个人信息的存储

2.1 存储期限：

我们仅在 "实现服务目的所必需的期限" 内存储您的信息：

  • Android ID、OAID、设备信息：存储至您的账号注销后 6 个月（用于后续账号找回及作弊追溯）；
  • 游戏时长、登录记录：存储至您的账号注销（用于防沉迷及运营分析）；
  • 若法律法规要求更长存储期限（如作弊违规记录需保存 1 年），则按规定留存。

2.2 停止运营通知：

若游戏停止运营，我们将提前 30 天通过游戏内公告、官网（若有）发布通知，并在停止运营后 15 个工作日内删除所有用户个人信息（或进行匿名化处理）。

三、信息安全

3.1 安全保护措施：

技术层面：采用 SSL 加密传输、设备信息脱敏处理（如 MAC 地址去标识化）、服务器防火墙等技术，防止信息泄露；

管理层面：严格限制访问用户信息的人员范围（仅授权必要工作人员），并签订保密协议，定期开展安全审计。

3.2 安全事件处置：

若发生信息泄露，我们将立即启动应急预案（如暂停相关功能、追溯泄露源头），并在 72 小时内通过游戏内推送、客服邮件告知您事件详情及补救措施；若难以逐一告知，将通过公告发布警示。

四、您的权利

4.1 信息查询与更正：

您可在游戏内【设置】-【账号安全】查询实名认证信息、设备绑定情况（含 Android ID、OAID 关联记录），并申请更正错误的实名信息（需提供有效证明）。

4.2 权限变更与账号注销：

您可通过设备【设置】调整权限授权状态（撤回授权后，对应功能无法使用，但不影响其他功能）；

若需注销账号（注销后所有信息将被删除，含 Android ID、OAID 关联记录），可发送邮件至 120613433@qq.com，提供账号绑定的手机号 / 邮箱，我们将在 3 个工作日内处理并回复。

4.3 投诉与建议：

若您对个人信息处理有疑问（如查询 SDK 收集的具体数据），可联系客服邮箱 120613433@qq.com，我们将在 5 个工作日内回复。

五、未成年人保护

5.1 监护人授权要求：

若您是未成年人（18 周岁以下），需在法定监护人同意后使用游戏服务并提供实名信息；监护人可联系客服查询 / 删除被监护人的游戏数据（需提供监护人身份证、未成年人身份证及监护关系证明，如户口本）。

5.2 防沉迷措施：

我们严格执行国家防沉迷政策：未成年人工作日每日游戏时长≤1.5 小时，法定节假日每日≤3 小时，22:00 - 次日 8:00 禁止登录；

若检测到未成年人疑似高额消费（如单次消费≥500 元），我们将通过短信联系监护人确认，未确认的消费将暂停到账并协助退款。
